# 📊 Customer Churn Analysis

## 🔍 Project Overview
An end-to-end data analysis project identifying why customers leave a telecom company and what business actions can reduce churn.

## 🛠️ Tools Used
- **Python** (Google Colab) — Data cleaning & EDA
- **Tableau Public** — Interactive dashboard
- **Libraries** — Pandas, Matplotlib, Seaborn

## 📁 Project Structure
| File | Description |
|------|-------------|
| `telco_churn.csv` | Raw dataset |
| `telco_churn_clean.csv` | Cleaned dataset |
| `Customer_Churn_Analysis.ipynb` | Python notebook |

## 📈 Key Findings
- Overall churn rate is **25%**
- **Month-to-month** customers churn at 38% — highest risk
- **New customers (0–15 months)** are most likely to leave
- **Price is NOT a churn driver** — both groups pay ~$68/month
- **Fiber optic** customers churn most despite premium service

## 💡 Business Recommendations
1. Incentivize month-to-month customers to switch to annual contracts
2. Launch onboarding program for new customers in first 3–6 months
3. Create loyalty program specifically for fiber optic customers
4. Focus retention budget on engagement — not price cuts
